# Camera Trap Analysis

A collection of R and Python scripts for processing, analysing, and visualising camera trap data.

---

## Overview

This repository contains workflows for camera trap data analysis used in wildlife monitoring and ecological research.

The scripts include data cleaning, detection summaries, occupancy preparation, activity pattern analyses, and visualization.



## Current Features

- Camera trap data cleaning
- Detection history generation
- Species detection summaries
- Activity pattern analysis
- Occupancy model preparation
- Visualization of camera trap results
